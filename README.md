# KSeF FakturaApp Presentation
Invoice management application written in Python, integrated with the Polish KSeF (National e-Invoicing System). It enables creation, validation, and submission of structured invoices.

## 📌 Overview
This project is an invoice management application designed to integrate with the Polish National e-Invoicing System (KSeF). It allows users to create, manage, validate, and submit structured invoices in compliance with official requirements.

The repository showcases how the application works, including basic integration with KSeF and handling of structured invoice formats.

## 🚀 Features
- Create and manage invoices
- Generate structured invoices compatible with KSeF (xml files)
- Validate invoice data before submission
- Integration with the KSeF API (authentication, document submission, status checking)
- User-friendly interface for invoice management
- Implemented in Python
